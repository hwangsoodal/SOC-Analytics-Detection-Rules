# SOC-Analytics-Detection-Rules

Этот репозиторий содержит набор простых и практических правил детектирования и аналитических заметок, которые помогают разбирать сетевую и системную активность с точки зрения SOC-анализа.

Правила для Suricata:
1. Детект ICMP Echo Request.
2. Детект подключения к SSH.
3. Детект подключения к RDP.
